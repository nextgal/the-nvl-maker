文件夹结构

THE NVL Maker解压之后，可以看到如下文件和文件夹：

  • ——包含吉里吉里的备用EXE、一些特殊功能扩展代码（插件）等，可以选择使用
  • ——工具启动需要的DLL
  • ——存放模板工程和游戏工程的地方
  • ——存放界面皮肤素材的地方
  • ——存放辅助工具的地方
  • ——存放教程（也就是本手册）的地方，请务必仔细阅读
  • ——启动THE NVL Maker
  • ——启动新建工程向导

project

project文件夹下面，还会包含若干个子文件夹。每个子文件夹都是一个游戏工程。
下载了NVL的范例之后，解压到这里就可以通过THE NVL Maker的编辑器打开查看了。
除了默认存在的Template（模板工程）以外，当开始制作游戏时，使用Wizard.exe（新建工程向导）创建出来的空白游戏也会生成到这里。
每个游戏本身的具体结构，请查看教程“THE NVL Maker游戏工程的结构”。

skin

skin文件夹是存放界面皮肤素材的地方，现在里面的界面素材都是由不同作者设计制作的。
具体作者请查看开发人员页面（同时包含了可以下载的PSD原文件地址）。
当你自己制作了界面素材和配置文件之后，也可以放到这里。
这样下一次新建工程时，就可以选择自己做的界面了（对界面排版等等没有大改动的系列游戏来说，这么做比从头开始更快）。

tool

tool文件夹下包含了吉里吉里/KAG官方工具的汉化版和NVL的辅助编辑工具。

  • ——编辑游戏设定文件Config.tjs的图形化工具，脱离NVL也可以单独使用
  • ——官方工具启动需要的DLL
  • ——打包时用到的krkr.exe，对需要打包成单个可执行文件，而非xp3的游戏，将会使用到这个exe
  • ——krkr.exe的编辑工具，可以用来修改图标，关闭控制台等，准备发布游戏时使用
  • ——文字图片包制作工具，具体使用方法请查看教程“预渲染文字（自定义字体）设定”
  • ——制作OGG循环标签文件（.sli）的工具，可以设定OGG格式的音乐以特殊的循环方式播放（例如播放一段前奏之后循环主体部分）
  • ——打包用工具，选择游戏工程下的Data文件夹之后，可以将游戏打包成单个exe或xp3资料档。具体使用方法请查看教程“游戏打包发布”
  • ——图片素材文件格式转换工具，可以在吉里吉里支持的几种图片格式中互相转换，例如BMP转PNG，PNG转TLG等
